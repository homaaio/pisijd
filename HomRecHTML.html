::: WHAT IS HOMREC? DETAILED DESCRIPTION :::

HomRec is a screen recording program created with one simple goal: to help people with weak computers record video without lags and slowdowns. If you have an old laptop, office PC, or just not very powerful hardware, and you've tried recording games or desktop with other programs (like OBS or Bandicam) but everything lagged ‚Äî HomRec is made for you!

ü§î Why does HomRec work on weak PCs?

It's simple: the program is written to use minimum CPU and RAM resources. It doesn't load the computer with unnecessary effects, doesn't require a powerful graphics card, and can adapt to your hardware. You can choose recording mode:

  • Ultra (60 FPS) ‚Äî for more or less modern computers, gives very smooth video;
  • Turbo (30 FPS) ‚Äî for very weak machines, less system load, but video still remains high quality.

üìπ What can you record?

  • Full screen;
  • Games (yes, HomRec can capture DirectX windows);
  • Webinars and online streams;
  • Tutorials and educational videos;
  • Calls in Skype, Zoom and other programs.

Important: In the current version, only video is recorded without sound. But I'm already working on adding sound recording from microphone and system in future updates!

üì¶ What's inside?

  • Simple and clear interface ‚Äî even a child can figure it out;
  • Three tabs: Performance (settings), Advanced (additional options), and Info (program info);
  • START, PAUSE, STOP buttons ‚Äî just like an old VCR :D
  • Live Preview ‚Äî you see what you're recording during capture;
  • Automatic video saving to "recordings" folder.

üíæ Technical details

  • Program size: only 120 MB (very small!);
  • Video format: .mp4 ‚Äî opens on any computer, phone, TV;
  • Supported OS: Windows 9, 10, 11 (32 and 64 bit);
  • In development: versions for macOS and Linux.

üõ°Ô∏è Antivirus complains? Don't worry!

Some antiviruses (especially Kaspersky and Avast) may complain about HomRec because the program is new and not yet well-known. It's not a virus! I've published the entire source code on GitHub, any programmer can check it. If your antivirus blocks the program ‚Äî just add the HomRec folder to antivirus exceptions. Everything will work immediately :)

‚ùì What is HomRec and what is it for?
HomRec is a program created for weak computers. If you have an old PC and your games lag during screen recording (FPS drops), HomRec helps. It uses few resources and works even on weak machines.
‚ùì On which computers does the program work?
On any computer with Windows 9, Windows 10 or Windows 11. Later we'll make versions for MacOS and Linux.
‚ùì How big is the program?
The program itself is about 120 MB. That's pretty small.
‚ùì How to install HomRec?
Very simple: download the ZIP file, unpack it and open the .exe file. Nothing complicated :D
‚ùì Antivirus complains about the program. Is it a virus?
No, it's NOT a virus! The entire source code is open on GitHub, you can check. Add the program to antivirus exceptions and everything will work.
‚ùì How to start screen recording?
Just click the "Start Recording" button. All recordings are saved in the "recordings" folder inside the program folder.
‚ùì What's the difference between Ultra and Turbo modes?
Ultra (60 FPS) ‚Äî for powerful computers, gives smoother video. Turbo (30 FPS) ‚Äî for weak PCs, less load on computer. If Ultra lags ‚Äî use Turbo.
‚ùì Can I record sound from microphone?
In current version sound is not recorded yet. I'm working on it, will add in future updates!
‚ùì What format are videos saved in?
Videos are saved in .mp4 format ‚Äî the most popular format that opens everywhere.
‚ùì What to do if the program lags?
Set lower settings (Turbo instead of Ultra), close unnecessary programs. If it doesn't help ‚Äî write to support.
‚ùì Why is there no sound in recording?
In this version sound is indeed not recorded. I know about it and am working to add sound in next versions.
‚ùì How to contact the developer?
Write to Telegram @homaexe or email homsupp@gmail.com. Source code on GitHub, you can create an Issue.
‚ùì Where to download new versions?
All new versions are immediately published in Telegram channel @homaexe and on GitHub. Subscribe to not miss updates!
‚ùì Will there be a version for macOS or Linux?
Yes, definitely! Currently the program is only for Windows, but I'm already working on Mac and Linux versions.
‚ùì How to help the project?
The best help is to recommend the program to friends, subscribe to the Telegram channel and star it on GitHub! This motivates to make HomRec better! Thanks ^^
‚ùì Is there a recording time limit?
No, the program can record as long as you want. But the longer the recording, the larger the file.
‚ùì Can I record only part of the screen?
This feature will appear in future updates! Follow news in Telegram.
